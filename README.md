# RAG Chat Bot Portal Integration

## Project Overview

RAG Chat Bot Portal Integration is an AI-powered chatbot solution designed to provide intelligent and context-aware responses using Retrieval-Augmented Generation (RAG). The chatbot integrates a custom knowledge base with modern Large Language Models (LLMs) to deliver accurate and relevant information.

The project demonstrates the integration of conversational AI, secure authentication mechanisms, and cloud deployment technologies for a scalable chatbot platform.

---

## Key Features

* Retrieval-Augmented Generation (RAG)
* Knowledge Base Integration
* Context-Aware Responses
* Secure HMAC-SHA256 Authentication
* Serverless Backend Functions
* Cloud Deployment using Vercel
* User-Friendly Web Interface

---

## System Workflow

1. User accesses the chatbot portal.
2. Secure authentication token is generated.
3. Chatbot initializes successfully.
4. User submits a query.
5. RAG engine retrieves relevant information from the knowledge base.
6. AI generates an accurate response.
7. Response is displayed to the user.
